기조연설 개요

Jensen Huang은 "Inference Inflection"이라는 핵심 프레임으로 기조연설을 열었다. AI가 단순한 질의응답 도구를 넘어, 지속적으로 작동하는 에이전트로 진화하는 시점에서 모든 생성 토큰이 컴퓨팅 경제의 기본 단위가 된다는 선언이다.

CUDA 20주년을 기념하며, cuDNN이 현대 AI의 "빅뱅"을 일으킨 순간을 회고했다. CUDA-X 라이브러리를 "왕관의 보석(Crown Jewel)"으로 표현하며 소프트웨어 생태계의 독보적 해자를 강조했다.

Jensen Huang은 AI 컴퓨팅 수요가 계속 폭증하는 이유를 3가지 동시 진행 스케일링으로 설명했다. 하나의 스케일링이 멈춰도 나머지 두 개가 계속 컴퓨팅 수요를 끌어올린다는 구조다.

① 사전학습 스케일링
더 크고 더 많은 데이터로 모델을 훈련. 데이터 양의 한계가 없으므로 학습 컴퓨팅 수요는 계속 증가.
② 사후학습(RL) 스케일링
RLHF, DPO 등 강화학습 기반 미세조정. 도메인 특화·정렬에 필요한 반복 학습 수요 폭증.
③ 추론(Inference) 스케일링
에이전트가 생각하고, 행동하고, 반성할수록 추론 컴퓨팅이 폭발. "AI has to think. In order to think, it has to inference."
BCG Insight — NVIDIA의 수직통합 전략
GTC 2026은 단순한 신제품 발표가 아니었다. 칩 → 소프트웨어 → 모델 → 에코시스템을 동시에 장악하는 4계층 수직통합 선언이었다. $1조 수주잔고에 "오히려 부족할 것"이라는 발언을 더해, 공급 부족을 전략적 자산으로 전환했다.
전략적 핵심
플랫폼 Lock-in 극대화. CUDA 20년 설치 기반 위에 Dynamo(추론 OS) + OpenClaw(에이전트 OS) + Nemotron(모델) + Jetson(엣지)까지 통합 → 경쟁사 칩으로의 전환 비용이 소프트웨어·모델·생태계 전체 교체 비용으로 확대.
경쟁 구도
AWS Trainium / Google TPU / MS Maia는 하드웨어 대안 → NVIDIA는 소프트웨어·모델 계층으로 전선 이동. AMD MI300X의 가격 경쟁력에 대응해 "CUDA 생태계 = 대체 불가"를 강조하는 포지셔닝 전략.
주요 리스크
하이퍼스케일러들의 내부 칩 비중 증가 추세 지속. Meta MTIA, Apple 자체 AI 칩 등 자사 워크로드 내재화 확산. 공급 부족 기간에 대안 개발이 가속되는 역설. 2027~2028년이 NVIDIA 지배력의 진짜 검증 구간.

플래그십

Vera Rubin은 단일 GPU가 아닌 총 7개 목적 특화 칩(아래 표 6개 + Blackwell Ultra 브릿지 포함)이 하나의 AI 팩토리를 이루는 통합 플랫폼이다. Jensen Huang은 키노트에서 "7 chips, 5 rack-scale computers"라는 구조를 직접 언급했다. Blackwell이 GPU 중심 설계였다면, Vera Rubin은 컴퓨트·네트워킹·스토리지·보안까지 공동 설계(Co-design)한 차세대 아키텍처다. NVIDIA 공식 발표 기준 Q1 2026에 양산 돌입했으며, H2 2026부터 파트너사(AWS, Azure, Google, Oracle) 배포 예정.

Dynamo 1.0은 추론 엔진(SGLang, TensorRT-LLM, vLLM)을 교체하는 것이 아니라 그 위에서 조율(Orchestration)하는 레이어다. 멀티 노드 환경에서 프리필·디코드 단계를 분리해 각각 독립적으로 스케일링한다.

SemiAnalysis InferenceX 벤치마크 기준 Blackwell 단독 대비 7x 처리량 향상. GB200 NVL72와 결합 시 최대 15x 복합 성능 향상. Llama 3.1 기준 TTFT(첫 토큰 생성 시간) 4x 감소, 처리량 1.5x 증가.

분리 추론 (Disaggregated Serving)
인코드·프리필·디코드 단계를 독립 워커로 분리. 각 단계별 최적 하드웨어 할당 → GPU는 프리필, Groq LPU는 디코드.
KV Block Manager (KVBM)
KV 캐시를 GPU 메모리 → CPU 메모리 → 로컬 SSD → 원격 스토리지(S3, Azure Blob) 계층 구조로 관리. CPU 임베딩 캐시로 TTFT 30% 감소.
NIXL (NVIDIA Inference Xfer Library)
NVLink를 통한 GPU 간 저지연 KV 캐시 전송 전용 라이브러리. DeepSeek v3 같은 대형 MoE 모델 로딩 7x 가속.
ModelExpress
GPU 간 모델 가중치 스트리밍으로 반복 스토리지 다운로드 제거. 멀티모달(이미지·영상) 워크로드 CPU 임베딩 캐시 통합.
BCG Insight — Dynamo 오픈소스 전략

전략적 핵심

주요 리스크

GR00T N2 — World Action Model
2026년 말 출시 · DreamZero 연구 기반
핵심 발표

GR00T N2는 World Action Model(세계-행동 모델) 아키텍처를 채택한 차세대 범용 로봇 기반 모델이다. 로봇이 환경을 내부적으로 시뮬레이션하며 행동 계획을 수립하는 구조로, 기존 비전-언어-행동(VLA) 모델의 한계를 돌파한다.

핵심 성과: 미지의 환경에서 새로운 태스크 완료 성공률이 최고 수준 VLA 모델 대비 2배 이상. 이는 로봇이 사전 학습되지 않은 작업을 실제 환경에서 수행할 수 있음을 의미한다.

BCG Insight — GR00T N2 전략적 의미
로봇 AI의 "ChatGPT 모먼트"가 시작됐다. 특정 태스크 전용 프로그래밍에서 범용 로봇으로의 패러다임 전환 — 현재 산업용 로봇 90%가 고정 프로그래밍 방식임을 감안하면, 유연 제조 시장의 구조적 재편을 예고한다.
전략적 핵심
NVIDIA는 GR00T(모델) + Cosmos(데이터) + Jetson/IGX(하드웨어)를 번들링해 로봇 AI 공급망 전체를 장악하는 구조. 로봇 제조사 입장에서는 별도 AI 파이프라인을 구축할 유인이 없어짐 — NVIDIA가 "로봇의 Android OS" 역할을 목표.
경쟁 구도
Physical Intelligence(π0)은 독립 기반 모델로 경쟁. Google DeepMind RT-2, Boston Dynamics Spot AI도 자체 생태계 구축 중. NVIDIA의 우위는 하드웨어(Jetson) + 시뮬레이션(Isaac Sim) + 모델(GR00T) 번들 — 개방형 생태계 전략으로 로봇 제조사 진입 장벽 낮춤.
주요 리스크
"2배 성공률" 주장은 외부 독립 검증 미완료. Sim-to-Real Gap — 시뮬레이션 환경에서 훈련된 모델이 실제 공장·물류 환경에서 동일 성능을 내지 못하는 문제는 학계에서 지속 보고됨. 상업 배포까지 검증 시간이 예상보다 길 수 있음.

DLSS 5: 3D-Guided Neural Rendering
2026년 가을 GeForce GPU 탑재
Game Changer

DLSS 5는 기존 딥러닝 슈퍼샘플링(DLSS)의 근본적 진화다. "3D-Guided Neural Rendering"이라는 새로운 패러다임을 도입하여, 구조화된 3D 데이터를 AI 생성 모델의 가이드로 사용해 포토리얼 조명과 재질을 구현한다.

핵심 작동 원리:

① 게임 엔진이 씬의 3D 구조 데이터(지오메트리, 노멀, 깊이 등)를 생성한다.

② AI 생성 모델이 이 3D 데이터를 가이드 신호로 받아, 렌더링되지 않은 디테일(조명 반사, 재질 질감, 캐릭터 세부 표현)을 예측·생성한다.

③ 패스 트레이싱(Path Tracing)이 실시간으로 모든 씬 요소에서 빛의 상호작용을 계산한다.

결과: 기존 래스터라이제이션 방식 대비 "포토리얼 조명"과 초고해상도 캐릭터 렌더링 달성.

Jensen Huang은 키노트에서 기업 AI의 핵심 원료로 "구조화 데이터(Structured Data)"를 전면에 내세웠다. SQL, Spark, Pandas, Velox 같은 데이터 프레임 플랫폼들이 처리하는 거대한 스프레드시트가 "비즈니스의 기준 진실(Ground Truth of Business)"이라고 정의했다.

Snowflake, Databricks, Amazon EMR, Azure Fabric, Google BigQuery 등 모든 주요 데이터 플랫폼이 이 구조화 데이터를 처리한다. NVIDIA의 cuDF 가속으로 이 워크로드를 5배 빠르게 하는 것이 곧 기업의 AI 추론 비용을 5배 낮추는 것과 동일하다는 논리다.

BCG Insight — 구조화 데이터 전략의 진짜 의미
Jensen Huang 원문: "Structured data is the foundation of trustworthy AI." — NVIDIA가 GPU 칩 기업에서 엔터프라이즈 데이터 인프라 전체 스택 기업으로 영역을 확장하겠다는 전략적 선언이다.
전략적 핵심
NVIDIA의 확장 공식 = "데이터 입력 → GPU 처리 → AI 출력" 전 사이클 통합. cuDF(정형 데이터 5배 가속) + cuVS(벡터 DB 12배 가속)로 데이터 전처리까지 GPU화 → 경쟁사가 AI 학습·추론 계층을 공략해도 데이터 파이프라인에서 NVIDIA 의존도 지속. Snowflake·Databricks 위에 NVIDIA가 앉는 구조.
파트너사 관계의 역설
Snowflake·Databricks는 협력사이자 잠재적 피식자. 현재는 cuDF 통합으로 성능을 높여주지만, NVIDIA가 데이터 웨어하우스 레이어까지 직접 진입 시 이해충돌 발생. 현재 파트너십은 NVIDIA에게 데이터 시장 정보와 진입 경로를 확보하는 전략적 탐색 단계로 해석 가능.

Insight 1. NVIDIA는 'AI 인프라 수직 통합자'로 진화 중

GTC 2026의 핵심 메시지는 단일 제품 발표가 아니다. NVIDIA는 칩(Rubin GPU + LPU) → 소프트웨어(Dynamo + CUDA) → 모델(Nemotron 6개 패밀리) → 에코시스템(Nemotron Coalition)의 전 레이어를 동시에 통제하는 수직 통합 전략을 완성하고 있다.

이는 과거 인텔의 x86 생태계 지배, 또는 애플의 하드웨어-소프트웨어-서비스 통합과 구조적으로 유사하다. 고객이 NVIDIA 스택의 어느 한 레이어라도 깊이 사용할수록, 전체 스택에서의 이탈 비용이 기하급수적으로 증가한다.

Insight 2. "추론 경제"의 도래 — 학습에서 추론으로 수익 무게중심 이동

Huang의 "Inference Inflection" 프레임은 단순한 마케팅 언어가 아니다. 현재 AI 모델 학습(Training) 시장은 GPT-4, Gemini 등 소수 대형 모델 기업이 주도하는 반면, 추론(Inference) 시장은 모든 기업과 소비자가 매 쿼리마다 사용하는 무한히 확장되는 시장이다.

Groq LPU의 인수 + Dynamo의 개발 + Vera Rubin의 추론 최적화는 NVIDIA가 "추론 경제"의 핵심 인프라가 되겠다는 선언이다. 조 단위 파라미터 모델이 에이전트로 24/7 운영되는 세계에서, 추론 비용의 1%를 줄이는 것이 학습 효율을 10% 높이는 것보다 더 큰 경제적 가치를 갖는다.

Insight 3. 자율주행 — '챗GPT 모먼트'의 의미와 현실적 간극

Huang의 "자율주행 ChatGPT 모먼트" 선언은 강렬하지만, 실제 상용화 일정(Uber LA 2027년)과 기술 성숙도를 냉정하게 평가해야 한다. ChatGPT가 2022년 11월 출시 후 1년 만에 1억 명을 달성한 것과 달리, 자율주행은 지역별 규제, 보험 체계, 인프라 정비라는 물리적 제약이 존재한다.

그러나 Waymo의 LA 확장, 중국 로보택시 시장의 폭발적 성장(BYD, Geely 참여)을 감안하면, 2027~2028년이 자율주행 상용화의 실질적 변곡점이 될 가능성은 높다. NVIDIA는 OEM과 로보택시 기업 양측 모두를 고객으로 확보한 유일한 플랫폼 사업자다.

Insight 4. 로봇공학 — 데이터 문제를 컴퓨팅 문제로 전환

산업용 로봇의 최대 병목은 데이터가 아니라 다양한 환경과 태스크에 대한 일반화 능력 부재였다. NVIDIA의 전략은 이를 "데이터 문제"로 재정의하고, Cosmos 세계 모델로 합성 데이터를 무한 생성하는 방식으로 해결한다.

이 전략이 성공하면, 로봇 훈련의 병목이 "실제 환경 데이터 수집"에서 "컴퓨팅 비용"으로 이동하게 된다. 컴퓨팅 비용은 NVIDIA가 가장 잘 통제할 수 있는 변수다.

Insight 5. $1조 수주잔고 — 신뢰 신호 vs. 실행 리스크

Jensen Huang이 공개한 Blackwell + Vera Rubin 합산 $1조 수주잔고(2027년까지)는 전년도 $5000억 예측이 실제로 초과 달성되고 있음을 시사하는 강력한 신뢰 신호다. 구형 Ampere 기반 GPU 클라우드 가격이 오히려 상승 중이라는 사실이 수요의 실질성을 뒷받침한다.

그러나 리스크 요인도 존재한다: ① TSMC 3nm 공급 제약, ② CoWoS-L 패키징 용량 한계, ③ 지정학적 수출 규제(중국 시장 배제), ④ AMD·Intel·자체 칩 경쟁의 장기적 부상. 수주잔고는 현재의 수요이지 미래의 보장이 아니다.

Insight 6. GTC 주가 반응 — "좋은 뉴스가 주가를 못 올렸다"

3월 16일 키노트 당일, NVIDIA 주가는 장중 최대 4.8% 급등 후 +1.7%($183.22)로 마감했다. 주가는 $1조 수주잔고 공개 직후 단기 급등 후 빠르게 되돌림. "좋은 소식이 지속적인 상승을 만들지 못하는" 패턴이 반복되었다.

월가의 신중론 이유:

  • Vera Rubin의 구체적인 단기 출하 일정과 수익 기여 시점에 대한 불확실성
  • AI 인프라 투자 버블 우려 — ROI 입증 압박 증가
  • 2027년부터 하이퍼스케일러 자체 ASIC(Trainium, TPU) 시장 침식 전망
  • 수출 규제로 중국 시장 배제 지속

Bank of America Vivek Arya는 목표가 $300 '매수' 유지. 38개 기관 중 38 Buy, 1 Hold. 애널리스트 컨센서스는 강세지만 시장은 신중.

Insight 7. CPU 시장 진입 — Intel·AMD와의 정면 대결

Vera CPU 88코어는 데이터센터 CPU 시장에서 Intel·AMD와의 직접 경쟁을 선언한 것이다. Mercury Research 기준 2025년 Q4 서버 CPU 시장 점유율: Intel 60%, AMD 24.3%, NVIDIA 6.2%.

Meta 등 대형 고객이 Vera CPU 랙(256 CPU/랙) 배포를 검토 중이라는 점은 의미심장하다. NVIDIA는 CPU로 서버 시장을 공략하는 것이 아니라, GPU 추론 파이프라인의 병목을 CPU 레이어에서 제거하기 위해 자체 CPU를 설계한 것이다. 이는 ARM 서버 CPU 시장 전반에 파급 효과를 줄 수 있다.

Insight 8. NemoClaw의 엔터프라이즈 수용의 진짜 맥락

기업들이 AI 에이전트 도입을 주저했던 이유는 기술 부재가 아니라 거버넌스·보안 문제였다. 실제로 Meta는 OpenClaw 사용 직원 디바이스에서 에이전트가 지시 없이 이메일을 대량 삭제한 사고 이후 사내 OpenClaw 사용을 제한했다.

NemoClaw가 해결하려는 핵심은: ① 프로세스 격리(에이전트 샌드박스), ② 최소 권한 접근 제어, ③ PII 자동 제거(외부 API 전송 전), ④ 감사 가능한 행동 로그. Gartner는 엔터프라이즈 AI 에이전트 시장이 2028년 $250억에 도달할 것으로 전망.

관련 핵심 논문
Attention Is All You Need (Transformer)
Vaswani et al. · Google Brain · NeurIPS 2017
GTC 2026에서 언급된 모든 언어 모델(Nemotron, Alpamayo 등)의 기반 아키텍처. NVIDIA Dynamo와 Groq LPU는 이 아키텍처의 디코드 단계 병목을 해결하기 위해 설계되었다.
Scaling Laws for Neural Language Models
Kaplan et al. · OpenAI · 2020
모델 크기, 데이터, 컴퓨팅의 스케일링 법칙. 조 단위 파라미터 모델의 이론적 근거. Vera Rubin의 "조 단위 파라미터 모델 수익 기회 10배" 주장의 배경.
DreamerV3: Mastering Diverse Domains through World Models
Hafner et al. · Google DeepMind · 2023
GR00T N2의 World Action Model 아키텍처와 직접 관련. 세계 모델이 로봇이 경험하지 않은 환경에서의 일반화를 어떻게 가능하게 하는지 이론적 기반 제공.
Sim-to-Real Transfer of Robotic Control with Dynamics Randomization
Tobin et al. · OpenAI · ICRA 2017
Omniverse 시뮬레이션에서 훈련된 로봇(Disney Olaf 포함)이 실제 물리 환경에서 동작하는 원리인 Sim-to-Real Transfer의 선구적 연구.
Neural Radiance Fields (NeRF)
Mildenhall et al. · UC Berkeley · ECCV 2020
DLSS 5의 3D-Guided Neural Rendering과 기술적 맥락 공유. 3D 씬 표현을 신경망으로 학습하는 접근법으로, DLSS 5의 3D 구조 기반 AI 생성과 개념적 연계.
UniSim: Learning Interactive Real-World Simulators
Yang et al. · Google · NeurIPS 2023
Cosmos 세계 모델과 유사한 접근법. 실제 세계의 물리적 상호작용을 학습하는 시뮬레이터 구축 연구. AV 및 로봇 훈련용 합성 데이터 생성의 이론적 토대.
AlphaFold 2: Protein Structure Prediction
Jumper et al. · DeepMind · Nature 2021
NVIDIA BioNeMo와 Google DeepMind의 단백질 데이터셋 협업 배경. AI 기반 단백질 구조 예측이 신약 개발 파이프라인에 미치는 영향.
